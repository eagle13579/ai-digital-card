# AI数字名片 北极星指标系统

## 北极星公式





北极星 = 活跃名片数 × 分享查看率 × 付费转化率

### 北极星公式拆解

北极星 = A × B × C

A = 活跃名片数（7日内有编辑/分享动作的名片数）
B = 分享查看率（分享Token被访问次数 / 分享总次数）
C = 付费转化率（付费用户 / 注册用户）

## 核心指标看板

| 指标 | 定义 | 采集方式 | 目标 |
|:-----|:------|:---------|:----:|
| DAU | 日活跃用户 | 登录事件 | TBD |
| 名片创建数 | 日创建画册数 | brochure create事件 | TBD |
| 分享率 | 创建者中分享的比例 | share_token生成事件 | TBD |
| 查看转化 | 分享页→注册的转化率 | share view + register | TBD |
| 付费率 | 注册→付费转化 | payment events | TBD |
| 留存率 | D1/D7/D30 | login events | TBD |

## 数据采集点

所有关键事件通过 MetricsMiddleware 自动采集：
- request_id + path + method + status + duration
- 存储在 events.db
