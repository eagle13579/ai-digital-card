### 2.3 数据流

```

   │
   ▼

   │  │  JWT 验证 (或 API Key 验证)                 │     │

   │                                                     │
   │  ┌── 业务处理 ────────────────────────────────┐     │
   │  │  Pydantic 输入校验                          │     │
   │  │  业务规则验证                               │     │
   │  │  数据库事务 (含乐观锁)                      │     │

响应 ← 结构化错误处理 ← 数据库/缓存/存储
```
