# k6性能基准测试报告

## 测试脚本

| 脚本 | 位置 | 场景 |
|:-----|:------|:------|
| smoke-test.js | k6/smoke-test.js | 冒烟测试(1VU) |
| load-test.js | k6/load-test.js | 负载测试(50VU) |
| stress-test.js | k6/stress-test.js | 压力测试(逐步到200VU) |

## 运行方式



bash
cd k6
k6 run smoke-test.js
k6 run load-test.js
k6 run stress-test.js



## 基准目标

| 指标 | 目标 | 警告线 | 危急线 |
|:-----|:----:|:------:|:------:|
| P95延迟 | <500ms | <1s | >2s |
| 错误率 | <0.1% | <1% | >5% |
| 吞吐量 | 1000 rps | 500 rps | <200 rps |
