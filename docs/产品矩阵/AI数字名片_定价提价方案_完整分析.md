### 2.1 Good-Better-Best 三阶（前台可见）

```
┌────────────────────────────────────────────────────────────┐
│                    Good → Better → Best                     │
│                                                             │
│  Free (L0)    Pro (L1-L2)    Oracle* (Decoy)    Enterprise  │
│  ¥0/月         ¥199/月         ¥299/月            ¥999/月    │

└────────────────────────────────────────────────────────────┘
                        ↑
               * 新增诱饵套餐
```

### 2.2 价值阶梯五层法（战略后台）

### 3.2 核心定价逻辑

#### Pro ¥199/月 — Why this price?

| 因子 | 计算 |
|------|------|
| **成本锚定** | 15个AI引擎 × 单引擎月均推理成本≈¥30-50 → 成本底线¥450+，¥199已是补贴价 |
| **竞品对标** | LinkedIn Premium ¥215/月，我们AI能力强3倍+，但定位非社交网络 |
| **划线价验证** | 原划线价¥199 → 直接转为实价，消除"虚标"不信任感 |
| **心理价位** | ¥199是中国人消费心理学中"200元以下"的最后防线（¥200+决策门槛陡升） |
| **年付等效** | ¥133/月 ≈ $18.6/月，仍比LinkedIn Pro低38%，性价比明确 |
| **预期ARR贡献** | 10,000付费用户 × ¥199 × 12 = ¥2,388万/年 |

#### Enterprise ¥999/月 — Why this price?

| 因子 | 计算 |
|------|------|
| **增量价值** | 团队协作+SSO+RBAC+API → 组织级价值至少是个人3-5倍 |
| **竞品对标** | Notion Business $18/用户/月，5人团队=$90/月≈¥648，但我们有AI引擎 |
| **利润空间** | ¥999/月含10人上限 → ¥99.9/人/月，远低于竞品$12-18/人/月 |
| **提价过渡** | ¥499→¥999 (+100%) 看似激进，但有¥1,999做锚定，¥999变成"合理" |
| **预期ARR贡献** | 500企业客户 × ¥999 × 12 = ¥599万/年 |

#### Oracle ¥299/月 — 诱饵套餐设计逻辑

| 机制 | 说明 |
|------|------|
| **定位** | 这是一个**故意设计得差**的套餐，目的是让Pro看起来更值 |
| **功能裁剪** | 与Pro相同名片数，但OCR降至300次/月，**无AI仪表盘**，**无水印去除**，**无智能匹配**，仅加一个无用的"金色虚拟徽章" |
| **价格对比** | ¥299比Pro贵¥100，但功能更少 → 任何理性用户都会选Pro |
| **行为经济学** | 经典"诱饵效应"：The Economist $59网页版/$125纸质版/$125纸质+网页的复制 |
| **实际购买** | 预计<1%用户选择此套餐，但会提升Pro转化率30-50% |

#### Enterprise Plus ¥1,999/月 — 至尊锚定

| 机制 | 说明 |
|------|------|
| **定位** | 不是普通套餐，是"定制企业方案"的标价锚点 |
| **功能** | 专属GPU推理集群 + 定制AI引擎微调 + 私有化部署 + 7×24专属AI军团支持 + 99.99% SLA |
| **效果** | 让¥999 Enterprise显得"性价比极高"，同时为真正的大客户定价谈判提供起点 |

### 3.3 Free 层收紧方案（推转化）

| 限制项 | 当前 | 收紧后 | 原因 |
|--------|------|--------|------|
| **名片数量** | 1张 | **1张** (不变) | 维持体验底线 |
| **OCR次数** | 3次/月 | **2次/月** | 刚好够体验，不够日常用 |
| **访客上限** | 5位/月 | **3位/月** | "看到有人看我的名片"但不够多 |

| **水印** | 有 | **有** (不变) | — |

| **导出功能** | 可用 | **禁用** | 免费不能导出联系人 |

> **预期效果：** Free→Pro 转化率从当前预估3-5%提升至**8-12%**。

## 六、Oracle 诱饵套餐的两种方案对比

### 方案A：保留Oracle诱饵（推荐⭐⭐⭐）

| 维度 | 评估 |
|------|------|
| **效果** | 经典诱饵效应已被大量SaaS验证，可提升Pro转化率30-50% |
| **风险** | 低。几乎无人购买，但需要额外维护一套定价逻辑 |
| **实施** | 灰度显示/50%透明度/加"参考对比"标签，避免用户困惑 |
| **退出机制** | 如3个月后<0.5%用户选择，可降级为"历史套餐"下线 |

### 方案B：去掉Oracle，用划线价替代

| 维度 | 评估 |
|------|------|
| **效果** | 简化页面，但失去诱饵效应的转化提升 |
| **替代方案** | Pro展示"¥299/月"划线价，暗示原价¥299现¥199 |
| **风险** | 划线价疲劳——太多SaaS用此手法，用户已脱敏 |
| **建议** | 仅作为A/B测试对照组 |

### 最终建议：**方案A**，Oracle ¥299作为诱饵，Pro ¥199作为主推，页面设计上Oracle用"对比参考"角标。

## 七、年付折扣策略

### 7.1 建议折扣结构

| 套餐 | 月付 | 年付(一次性) | 等效月价 | 折扣率 | 行业对标 |
|------|------|-------------|----------|--------|----------|
| Pro | ¥199/月 | ¥1,590/年 | ¥133/月 | **~33%** | Linear, Notion: 15-20% |
| Oracle | ¥299/月 | ¥2,390/年 | ¥199/月 | ~33% | — |
| Enterprise | ¥999/月 | ¥7,990/年 | ¥666/月 | ~33% | Figma: 17-20% |
| Enterprise Plus | ¥1,999/月 | ¥15,990/年 | ¥1,333/月 | ~33% | 定制 |

> **为什么给33%而不是行业标准的20%？**
> 1. 我们目前是**从零建立年付习惯**，更大的折扣激励用户"改习惯"
> 2. 33%折扣使年付等效月价≈¥133，恰好低于¥200心理线，大幅降低决策阻力
> 3. 12个月锁定大幅降低流失率（预计年付用户留存率95%+ vs 月付75-80%）
> 4. **6个月后可调整为20%**（标准行业水平），初期用33%打市场
