## 试用转化漏斗

### 免费 → 试用 → 升级 完整流程

```
┌──────────┐    ┌──────────────┐    ┌──────────────┐
│  免费版    │ ──→│ 14天免费试用  │ ──→│  标准版/企业版 │
│  Free     │    │  Standard    │    │  Paid Plan   │
│  ¥0       │    │  ¥0 (试用期)  │    │  ¥99/¥499    │
└──────────┘    └──────────────┘    └──────────────┘
     │                  │                   │
     │ 注册即享          │ 一键开通           │ 支付后永久使用
     │ 基础功能          │ 全部标准版功能      │ 按需选择套餐
     └──────────────────┴───────────────────┘
```

### 转化节点

1. **注册激活** — 用户注册后自动获得免费版
2. **引导试用** — 在免费版界面引导用户点击"免费试用14天"
3. **试用期间** — 全功能体验标准版，倒计时提醒剩余天数
4. **试用到期前** — 到期前3天、1天发送转化提醒
5. **升级转化** — 一键升级到标准版（¥99/月）或企业版（¥499/月）
6. **降级处理** — 用户可选择降级回免费版，当前周期结束后生效

### 试用规则

- 每个用户 **仅限一次** 14天免费试用
- 试用期间享受标准版全部功能
- 试用到期自动转为免费版（如需升级需手动操作）
- 试用期间可随时升级为正式付费用户
- 试用结束转为免费版后，超出免费版限制的功能将暂停

## 常见问题

**Q: 试用到期后我的数据会丢失吗？**
A: 不会。试用到期后数据会保留，但超出免费版限制的功能（如无限访客记录、信任网络等）将暂停。续费后即刻恢复。

**Q: 能否从企业版降级到标准版？**
A: 可以。降级在当前计费周期结束时生效，当前周期内您仍可使用企业版全部功能。

**Q: 是否支持年付？**
A: 当前仅支持月付。年付方案即将推出。

**Q: 免费版有多少次匹配机会？**
A: 免费版每日匹配次数有限制（具体以系统实际配置为准），标准版和企业版不限匹配次数。
