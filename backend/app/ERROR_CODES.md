# AI数智名片 — 统一错误码规范

> 版本: v1.0 | 创建: 2026-07-10

## 错误码格式
`XXX-NNNN` 其中 XXX=模块, NNNN=4位数字

## 模块前缀
| 前缀 | 模块 |
|------|------|
| ATH | 认证(Auth) |
| USR | 用户(User) |
| BRC | 画册(Brochure) |
| CRM | 客户管理 |
| PAY | 支付(Payment) |
| AI_ | AI能力 |
| MAT | 匹配(Match) |
| TNT | 多租户(Tenant) |
| SEC | 安全(Security) |
| GEN | 通用(General) |

## HTTP状态码映射规则
| 错误类型 | HTTP状态码 | 说明 |
|---------|:----------:|------|
| 参数错误 | 400 | 请求参数校验失败 |
| 未认证 | 401 | 缺少或无效token |
| 无权限 | 403 | RBAC拒绝 |
| 资源不存在 | 404 | 资源未找到 |
| 冲突 | 409 | 资源已存在 |
| 频率限制 | 429 | 限流触发 |
| 服务端错误 | 500 | 内部错误 |
| AI服务不可用 | 503 | AI降级中 |

## 标准响应格式
```json
{
  "code": "GEN-0001",
  "message": "人类可读的错误描述",
  "detail": "调试详情(仅开发环境)",
  "request_id": "req-xxx"
}
```
