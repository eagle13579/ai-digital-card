## 二、已沉淀的最佳实践

### 2.1 Two-Tower + Hard Negative Sampling（备忘：适用于100+用户场景）

**原则**: 小数据阶段(<100用户)以规则引擎为主，不训练端到端DNN模型

**适用条件**:
- 用户数 ≥ 100 且有足够的负样本 (推荐正:负=1:3~1:5)
- 标签系统成熟，标签覆盖率 ≥ 70%
- 有明确的匹配反馈闭环 (like/dislike/click)

**架构推荐**:

```python
# 伪代码 — Two-Tower匹配模型
class TwoTowerMatcher:
    def __init__(self):
        self.user_tower = DNN(128→64→16)  # 用户编码
        self.card_tower = DNN(128→64→16)  # 名片编码

    def forward(self, user_features, card_features):
        user_emb = self.user_tower(user_features)
        card_emb = self.card_tower(card_features)
        return cosine_similarity(user_emb, card_emb)
```
