# AI数字名片 · 20人军团验收主控方案

## 基本信息

| 项目 | 内容 |
|:-----|:------|
| 验收目标 | AI数字名片全功能验收（前端35页面 + 后端258API） |
| 测试环境 | localhost:8201 (FastAPI后端) |
| 测试日期 | 2026-07-13 |
| 总指挥 | 白泽 |
| 验收方法 | 20人数智员工并行测试 |

## 三层验收架构

```
┌─────────────────────────────────────────────────┐
│          Layer 1: 用户角色视角（8人）                │
│  每种人设独立走完整用户旅程                          │
├─────────────────────────────────────────────────┤
│          Layer 2: 模块专项测试（8人）                │
│  每个模块/页面逐功能深度测试                         │

│          Layer 3: 压力·安全·数据（4人）              │
│  后端渗透/合规/数据/性能                           │
└─────────────────────────────────────────────────┘
```
