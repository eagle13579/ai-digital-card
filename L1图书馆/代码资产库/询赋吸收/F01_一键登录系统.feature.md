# F01 — 一键登录系统

## 心智模型（Mental Model）

用户"一键登录"的本质不是"登录"，而是**手机号即身份**。系统将手机号作为唯一标识，新用户无需注册流程，输入手机号即自动完成账号创建。这消除了传统注册的门槛（填写用户名、密码、邮箱等），实现**零摩擦获客**。

```
用户输入手机号
  → 后端查询 users 表
    → 存在 → 生成 JWT Token → 返回登录状态
    → 不存在 → 自动 INSERT 用户记录 → 初始化 privacy_settings → 生成 JWT Token → 返回注册+登录状态
```

## Architecture Decision Records

### ADR-001: 选择"手机号+JWT"而非第三方OAuth

| 项目 | 决策 |
|------|------|
| 上下文 | 询赋面向商务社交场景，目标用户群体习惯手机号验证 |
| 决策 | 使用手机号作为主键标识，JWT 承载认证态 |
| 理由 | 微信小程序生态中手机号是最普适的标识；无需对接多个OAuth平台降低复杂度 |
| 后果 | 需处理短信验证码发送（可接入阿里云/腾讯云 SMS） |

### ADR-002: 自动注册 + Token 生产一体化

| 项目 | 决策 |
|------|------|
| 上下文 | 用户不需要"注册"步骤，输入手机号即完成全流程 |
| 决策 | 登录接口内判断用户是否存在，不存在则自动注册，返回 `isNew: boolean` 标记 |
| 理由 | 前端无需维护注册/登录两套 UI 和状态机，一套 `/login` 搞定 |
| 后果 | 接口返回需携带 `isNew` 标识，前端据此决定是否跳转引导页面 |

### ADR-003: Token 有效期 7 天 + 无刷新机制

| 项目 | 决策 |
|------|------|
| 上下文 | 商务社交场景不需要高频安全验证 |
| 决策 | JWT expiresIn = '7d'，客户端 localStorage 持久化 |
| 理由 | 简化前后端实现；7天足够覆盖商务活动周期 |
| 后果 | 不支持多端互踢；Token泄露有7天窗口期 |

## 核心代码提取

### 后端 authService.ts

```typescript
// 一键登录+自动注册 核心逻辑
static loginByPhone(phone: string): { token: string; userId: string; isNew: boolean } {
  let user = db.prepare('SELECT id FROM users WHERE phone = ?').get(phone);
  let isNew = false;
  if (!user) {
    const userId = uuid();
    db.prepare('INSERT INTO users (id, phone, nickname) VALUES (?, ?, ?)')
      .run(userId, phone, `用户${phone.slice(-4)}`);
    db.prepare('INSERT INTO privacy_settings (user_id) VALUES (?)').run(userId);
    user = { id: userId };
    isNew = true;
  }
  const token = generateToken({ userId: user.id, phone });
  return { token, userId: user.id, isNew };
}
```

### JWT 中间件 auth.ts

```typescript
// JWT Token 生成与验证
const JWT_SECRET = process.env.JWT_SECRET || 'xunfu-secret-key-2024';

function generateToken(payload: AuthPayload): string {
  return jwt.sign(payload, JWT_SECRET, { expiresIn: '7d' });
}

// Express 中间件
function authMiddleware(req, res, next) {
  const authHeader = req.headers.authorization;
  if (!authHeader?.startsWith('Bearer ')) {
    return res.status(401).json({ code: 401, message: '未登录或登录已过期' });
  }
  try {
    req.user = verifyToken(authHeader.substring(7));
    next();
  } catch {
    res.status(401).json({ code: 401, message: 'Token 无效或已过期' });
  }
}
```

### 前端 Axios 拦截器

```typescript
// 请求拦截 - 自动注入 Token
api.interceptors.request.use((config) => {
  const token = localStorage.getItem('token');
  if (token) config.headers.Authorization = `Bearer ${token}`;
  return config;
});

// 响应拦截 - 401 自动登出
api.interceptors.response.use(
  (res) => res.data,
  (err) => {
    if (err.response?.status === 401) {
      localStorage.removeItem('token');
      window.location.hash = '#/login';
    }
    return Promise.reject(err.response?.data || { message: '网络错误' });
  }
);
```

## 数据库 Schema 映射

```sql
CREATE TABLE users (
  id TEXT PRIMARY KEY,
  phone TEXT UNIQUE NOT NULL,
  nickname TEXT,
  avatar TEXT,
  company TEXT,
  position TEXT,
  industry TEXT,
  city TEXT,
  created_at DEFAULT unixepoch(),
  updated_at DEFAULT unixepoch()
);

CREATE TABLE privacy_settings (
  user_id TEXT PRIMARY KEY REFERENCES users(id)
  -- 各字段为布尔值，控制信息可见性
);
```

## 可复用程度

- **后端逻辑**: 90% 可直接复用（仅需适配数据库引擎）
- **前端逻辑**: 70% 需适配微信小程序 API
- **架构模式**: 100% 可复用
