## 一句话定位
定时自动扫描数据库，检查数据完整性和一致性

## 能力
- 6类检查: 邮箱格式/评分范围/分级有效性/日期合理性/外键完整性/孤立数据
- JSON报告输出到 logs/data_quality_report.json
- exit code = 违规数, 可集成到CI/CD
- 可作为定时任务: 每天凌晨执行

## 适用产品
所有有SQLite/关系型数据库的产品

## 使用方式
```bash
python scripts/data_quality_check.py
echo $?  # 0=干净, >0=违规数
```
