## 一句话定位
订单创建→支付回调→订阅激活全链路骨架

## 能力
- 3档定价: basic(¥999)/standard(¥1999)/flagship(¥3999)
- 订单创建+支付处理(mock实现,可替换真实网关)
- 订阅管理(激活/过期/自动续费)
- 订阅状态查询

## 适用产品
所有SaaS订阅制产品

## 使用方式
```python
from api.services.payment_service import create_order, process_payment, get_subscription
order = create_order(user_id, tenant_id, "standard")
result = process_payment(order["order_no"], method="wechat")  # 替换为真实网关
sub = get_subscription(user_id)
```

## 部署要求
- payment_orders + subscriptions 两张表
- 接入真实支付网关前需要商户号
