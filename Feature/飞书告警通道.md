## 一句话定位
服务宕机/异常时通过飞书机器人推送告警

## 能力
- send_feishu(title, message, level) → 推送飞书消息
- send_alert() 四通道: 飞书+Sentry+日志+文件fallback
- 5级告警: debug/info/warning/error/critical
- 失败不阻断,静默fallback

## 适用产品
所有需要监控告警的产品

## 使用方式
```python
FEISHU_WEBHOOK=https://open.feishu.cn/open-apis/bot/v2/hook/xxx python app.py
# 或 .env 中设置
```

## 部署要求
- 飞书机器人Webhook URL
- requests库
