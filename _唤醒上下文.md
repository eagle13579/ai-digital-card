## 一句话定位

AI数字名片是链接链客宝企业家生态圈的智能电子名片系统，让企业主一分钟创建个人/企业数字画册，通过名片匹配+信任网络实现精准商业连接。

## 架构（五种模式）

```
Gateway(链客宝:5136) → 统一API入口
delegate(匹配引擎/同步/OCR) → 子任务委派
Profiles(:8003独立服务) → 7x24值守
execute(数据聚合/cron/cronjob脚本) → 机械层
Kanban(北极星看板/迭代管理) → 长期任务调度
```
