## 选择标准
1. AICPA认证的CPA firm
2. 有SaaS/API产品审计经验
3. 接受远程审计
4. 报价包含Type I + Type II

## 准备工作
1. 确定审计范围(系统边界)
2. 确定信任准则(Security + Availability + Confidentiality)
3. 选择审计类型(Type I先, Type II后续)
4. 签署NDA + 服务合同
