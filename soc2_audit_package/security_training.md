# 安全培训计划

> 版本: v1.0 | SOC2 REF: DOC-05
> 创建日期: 2026-07-10

## 1. 培训对象
- 所有开发人员(必须)
- 运维人员(必须)
- 产品经理(推荐)

## 2. 培训内容
| 模块 | 内容 | 时长 | 频率 |
|------|------|:----:|:----:|
| 安全编码 | OWASP Top 10 / 输入校验 / SQL注入防护 | 2h | 入职+每年 |
| 数据保护 | 数据分类/加密/脱敏/最小权限 | 1h | 入职+每年 |
| 事件响应 | 发现→报告→止损→复盘完整流程 | 1h | 每半年 |
| AI安全 | Prompt注入/模型输出校验/数据泄露风险 | 1h | 每季度 |

## 3. 考核方式
- 入职培训后通过安全测试(80分及格)
- 未通过者不得操作生产环境
- 年度培训记录留存至SOC2审计
